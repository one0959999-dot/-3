# 시나브로 (Sinabro)

> 시나브로: 모르는 사이에 조금씩 — 토스증권 API 기반 무인 자동투자 봇.

## 전략 (교과서 v3, 동결)
- **국내**: KODEX200 지수 ETF 50% + v3 저변동 25종목 50% (동일비중, 분기 리밸런스)
  - 선정: 126일 저변동 상위 + 상승중인 200일선 위 + 부실(자본잠식·2년적자)·정체가격·거래일부족 제외
  - **참고서 레이어**: 데이터 아티팩트·상폐위험 종목 사전 제외 (reference_data/)
  - **지수 DCA**: 대형 목돈 유입시 지수 몫만 6개월 분할 진입 (고점 몰빵 회피)
  - 금지: 시장타이밍·손절·리스크오프 오버레이 (검증에서 전부 수익 파괴)
- **미국**: USD 예수금 생기면 SPY 매수 (원화 자동환전 없음 — 수동 환전 필요)

## 구성
- `KR/algo_v1.py` — 종목선정 단일소스 (백테스트=라이브 동일 함수)
- `KR/live_v1.py` — 유니버스·목표 산출
- `KR/auto_order.py` — 신규자금 배분(buy-only) + 분기 완전자동 리밸런스 + 지수 DCA
- `KR/auto_deploy.py` — 현금 감지 자동투입 (크론 매일)
- `KR/auto_order_us.py` — 미국 SPY 자동매수
- `KR/deadman.py` — 감시장치 (봇 정지 알림)
- `KR/reference.py` + `reference_data/` — 참고서 (아티팩트·상폐 회피)
- `base/app.py` — 시나브로 대시보드 (조회 전용, 주문 없음)
- `base/toss_api.py` — 토스증권 OpenAPI 실행층

## 운영 (EC2)
- 크론(KST): 평일 10:00 리밸런스 · 10:30 신규자금 배분 · 미국장 아침 US · 매일 감시
- 배포: 로컬 커밋·푸시 → EC2 `git pull` → `sudo systemctl restart lassibot` (대시보드)
- `lassi.db`는 gitignore — 수동 전송. 최신 운영 상태는 `HANDOFF.md`.
