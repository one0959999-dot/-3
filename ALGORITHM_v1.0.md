## 0. 정답지 방법론 (확정)
- **정답지 = 이상적 매매 타이밍** = 사후로 본 바닥(매수)·천장(매도) 전환점 + 그 사이 9국면.
- 생성: 종목별 **zigzag(되돌림 20%)** 전환점 → 레그를 **시간비율**로 9국면 라벨(회복초입/상승초입/중반/말기, 하락초기/중반/말기, 패닉, 횡보).
- 기간: **2014~2025** (2026 데이터 손상으로 컷). 지수 + 종목별(상폐 포함).
- 각 구간 근거(RSI·200MA이격·모멘텀·거래량) 첨부 → 검증 가능.
- 산출물: `KR/answer_sheet.py`, `answer_sheet_stocks.py`, `answer_key_detail.py`, `plot_answersheet.py`

## 1단계 (★확정): 국면 판단 = **봇(규칙기반)**

### 결정
**국면 9단계 판단은 봇(classify_phase 트리)으로 한다.** AI는 국면 판단에 쓰지 않는다.

### 근거 (15개 동일종목, 정답지 대비 일치율 vs 랜덤)
| 단계 | 봇 | AI(가격+정세+금리) | 봇+AI | 랜덤 |
|---|---|---|---|---|
| 9단계 | **18%** | 7% | 10% | 11% |
| 5단계 | **33%** | 19% | 21% | 20% |
| 3단계(상승/하락/횡보) | **48%** | 31% | 31% | 33% |

- **봇: 모든 단계서 랜덤 초과 = 실력 있음.**
- **AI: 모든 단계서 랜덤 이하 = 국면감지 능력 없음** (금리·정세 줘도, 날짜익명 조건). 거의 '횡보'로만 답함.
- **봇+AI: AI가 봇을 랜덤으로 끌어내림** → 결합 안 함.

### 봇 국면판단 규칙 (실시간·워크포워드, 룩어헤드 없음)
MA200 대비 위치·기울기, 20/60일 모멘텀, ADX, 52주고점, (지수는 VIX) →
패닉/하락초기/중반/말기 · 회복초입/상승초입/중반/말기 · 횡보.
산출물: `KR/detect_realtime_bot.py` (`bot_realtime_phase`).

### 한계 (정직)
- 봇도 절대정확도는 낮음(9단계 18%) — 실시간은 미래 전환점을 모르니 늦고 노이즈. 단 **랜덤보다 일관되게 나음**이 핵심.
- 3단계(방향) 48%가 실용 기준선. 이걸 다음 단계(매매)에서 활용.

- ❌ 국면판단/타이밍/전환점: 봇·랜덤보다 못함 (정량 영역은 규칙/퀀트).

  (상세 의견은 대화 기록 참조)

## 향후 로드맵 (누적 의존 — 각 단계는 앞 단계 확정결과를 적용 후 진행)

```
v1.0 [확정] 국면판단 = 봇
  ├─ ① 섹터별 사이클 파악       ← v1.0 국면판단을 '섹터'에 적용
  ├─ ② 국면별 매매방식 결정      ← v1.0 국면 위에서 무엇을 살/팔지 (정답지 '이상대처' 검증)
  │        ↓ (①+② 적용)
  ├─ ③ 섹터별 포트폴리오 구성     ← 섹터사이클 + 매매방식 합쳐 배분 (②완료 후)
  │        ↓ (③ 적용)
  └─ ④ 섹터별 리밸런싱 (언제/어떻게)  ← 포트폴리오를 언제·어떻게 갈아탈지
```

| 순서 | 내용 | 전제(적용할 앞단계) |
|---|---|---|
| **①** | 섹터별 사이클(로테이션) 파악 | v1.0 국면판단 |
| **②** | 국면별 매매방식(매수/보유/매도/현금·투입비율) | v1.0 국면판단 |
| **③** | 섹터별 포트폴리오 구성 | ① 섹터사이클 + ② 매매방식 |
| **④** | 섹터별 리밸런싱 시점·방법 | ③ 포트폴리오 |

★누적 원칙 (덮어쓰기 아님, 쌓기):
```
v1.0 국면판단 = 모든 단계의 공통 토대
 ① 실행
 ② = ① 누적           (v1.0 + ① + ②)
 ③ = ①+② 누적         (v1.0 + ① + ② + ③)
 ④ = ①+②+③ 누적      (v1.0 + ① + ② + ③ + ④ = 완성 알고리즘)
```
→ **최종 알고리즘 = 앞 단계 전부 합쳐진 하나.** 각 단계는 이전 단계 결과를 실제 적용한 상태에서 검증·확정(독립실험 아님).
보조: AI는 국면판단 아닌 **③ 종목선정의 정성 리스크필터(공시·뉴스로 부실종목 제거)** 후보로 검증 예정.
